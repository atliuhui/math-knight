# Math Knight

Math Knight 是一个面向口算练习的回合制数学战斗游戏。玩家选择存档和难度后进入战斗，每轮回答一道小学数学题；答对会攻击 Boss，答错会受到伤害。

## 玩法

- 玩家初始生命值为 10，Boss 默认生命值为 20。
- 每局最多 20 轮，Boss 生命值归零则胜利，玩家生命值归零则失败。
- 每轮显示上一题、当前题和下一题，提交答案后题目队列向上推进。
- 答对当前题：Boss 受到 1 点伤害，并记录本题有效用时。
- 答错当前题：玩家受到 1 点伤害，错误题不会计入有效题数和有效用时。
- 可随时暂停战斗，暂停时间不会计入当前题用时。

## 难度

首页提供 L01-L20 难度，覆盖整数加减、乘除口诀、小数加减、最大公因数和最小公倍数等练习。难度卡片可以横向滑动，触屏设备上支持滑动和单击选择。

已达到高分记录的难度会恢复 Boss 图像颜色；未掌握的难度使用灰度图像提示。

## 操作

1. 在首页选择一个存档。
2. 选择一个难度。
3. 点击“开始战斗”。
4. 使用游戏内数字键盘输入答案。
5. 点击右侧或下方的提交区域提交答案。

游戏不依赖系统键盘，适合桌面浏览器和手机 H5 页面操作。

## 统计

战斗结束后会显示本局统计和历史记录，包括：

- 本局积分
- 正确率得分和用时得分
- 正确题数、错误题数和最长连击
- 总有效用时、平均用时、最快用时和最慢用时

单局积分由答题、用时和连击共同获得：

```text
本题得分 = 答题分 + 用时分 + 连击分
答题分 = 答对 +1，答错 +0
用时分 = ≤3 秒 +3，≤6 秒 +2，≤10 秒 +1，>10 秒 +0
连击分 = 当前连击数 - 1
```

每当本局积分首次达到或跨过 27 的倍数，触发一次随机奖励。若触发时同时达到该 27 倍数之后的第一个质数，则本次奖励升级为高级奖励。

一般奖励包括：

- 抵消伤害一次。
- 保护连击数量一次。
- 时间停滞：下一题用时分至少获得 2 分。

高级奖励包括：

- 在答案框中给出第一个数字。
- 允许重新答题一次。

## 徽章

每个 Level 独立收集一套徽章。L01-L20 各有 7 枚成语徽章，总计 140 枚。

| 徽章 | 条件 |
| --- | --- |
| 势如破竹 | 该 Level 单局最大连击 >= 10 |
| 百发百中 | 该 Level 单局准确率 = 100% |
| 一触即发 | 该 Level 单局最小用时 <= 3 秒 |
| 风驰电掣 | 该 Level 单局平均用时 <= 5 秒 |
| 持之以恒 | 该 Level 连续练习 >= 20 天 |
| 日积月累 | 该 Level 累计练习 >= 50 天 |
| 万夫莫敌 | 该 Level 单局完胜：Boss HP 归零且玩家存活 |

## 存档

游戏支持 5 个玩家存档。玩家名称、当前选择和统计历史会保存到 Godot 的 `user://` 目录。当前应用用户目录名为 `math-knight`。
